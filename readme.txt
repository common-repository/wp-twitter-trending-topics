== Description ==

Este plugin te ayudar&aacute; a mostrar los twitter trend topics actuales de Twitter por regi&oacute;n.

Una manera de proporcionar a nuestros visitantes de nuestro sitio web, de forma sencilla, los Twitter Trending Topcis del momento. Este plugin te ayudar&aacute; a mostrar los twitter trending topics actuales por Regi&oacute;n en tu blog de WordPress.


<strong>C&oacute;mo te ayuda este plugin?</strong>

* Muestra a tus visitantes los twitter trending topic actuales.
* Puedes mostrar los twitter trending topics actuales por regi&oacute;n seleccionada.
* Te ayudara a aumentar tus seguidores en Twitter.


== Installation ==


Pasos de la instalaci&oacute;n

1. Sube WPTwitter-Trending-Topics en tu carpeta <strong>/wp-content/plugins/</strong> via FTP. O tipea <strong>Twitter Trendings Topics</strong> por el men&uacute; de plugin de WordPress.
2. Activa el plugin a trav&eacute;s del men&uacute; '<strong>Plugins</strong>' en tu WordPress
3. A&ntilde;adir el widget a la barra lateral deseada en el men&uacute; de WordPress Widgets.
4. Selecciona tu regi&oacute;n, seleccione la hora de actualizaci&oacute;n autom&aacute;tica y el n&uacute;mero de las twitter trending topics a mostrar.
5. Califica el plugin y comprueba que funciona en wordpress.org.
6. Deja un comentario acerca del plugin en <strong>http://www.laliamos.com/wordpress-twitter-trending-topics-plugin/</strong>

Algunas sugerencias de formato

* Si sabes CSS entonces puedes cambiar esta combinaci&oacute;n de colores en tu widget de acuerdo a tu blog en WordPress.
